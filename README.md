# Robin D. Estiny
### A discord.py bot connected to the Bungie.Net API

## Features:
- Getting specific player information about their characters and platform
- Getting weekly grandmaster nightfall information

## .env file keys
- DESTINY_API_KEY
- DISCORD_API_KEY
- CLIENT_ID
- CLIENT_SECRET
- MEMBERSHIP_TYPE
- MEMBERSHIP_ID
- HUNTER_ID
- WARLOCK_ID
- TITAN_ID
